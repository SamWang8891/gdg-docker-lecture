node_modules left behind.

這頁是 Vite build 完的靜態檔，由 nginx 服務。runtime image 沒有 Node、沒有 npm、沒有 source。

Dockerfile · two stages nginx:alpine · :80
stage 1

builder FROM node:20-alpine

  • npm ci → node_modules (≈ 200 MB)
  • npm run build → dist/
  • 整個 Node runtime + 工具鏈
build 完整個 stage 被丟掉
COPY --from → dist/
stage 2

runtime FROM nginx:alpine

  • COPY --from=builder /app/dist
  • nginx 服務靜態檔
  • 沒有 Node，沒有 npm，沒有 source
↑ 這就是你現在看到的 image
